How to run the program:

Use the list of genres given to pick a desired genre of movie. 
Then, input the year that you want the movies to be released in. 
Finally, input the maximum runtime you want the movie to be.

The top five movies on Letterboxd that match your desired search terms will be given. The list will show the name of the movie, its rating, its runtime, and a brief description of what it is about.
